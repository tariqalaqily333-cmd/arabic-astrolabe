# مولد الأسطرلاب | Astrolabe Generator

مولد أسطرلاب مسطح كامل مكتوب بلغة Python. يرسم جميع أجزاء الأسطرلاب الكلاسيكي بدقة: الأم، الصفيحة، العنكبوت، الظهر، والمسطرة. يعتمد على الإسقاط المجسم الصحيح رياضياً.

## ✨ المميزات

- **دعم كامل للعربية**: جميع النصوص والأرقام باللغة العربية مع استخدام خط Amiri
- **دقة فلكية عالية**: استخدام الإسقاط المجسم Stereographic projection الصحيح
- **قابل للتخصيص بالكامل**: يمكنك تحديد خط العرض، والنجوم، والتاريخ والوقت بسهولة
- **جاهز للطباعة 4K**: يولد صور بدقة 2480×2480 بكسل مناسبة للطباعة عالية الجودة
- **جميع الأجزاء**: الأم، صفيحة واحدة، العنكبوت، الظهر، والمسطرة
- **بدون مكتبات معقدة**: يعتمد فقط على مكتبة `Pillow` لتوليد الصور
