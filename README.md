# Astrolabe Generator الأسطرلاب

مولد أسطرلاب عربي كامل بالـ Python. يرسم كل أجزاء الأسطرلاب الكلاسيكي بدقة: الأم، الصفيحة، العنكبوت، الظهر، والعضادة.

## المميزات
- **دعم عربي كامل**: كل النصوص والأرقام بالعربي باستخدام خط Amiri
- **دقة فلكية**: إسقاط مجسم Stereographic projection صحيح رياضياً
- **قابل للتخصيص**: حدد خط العرض والنجوم اللي عايزها
- **4K جاهز**: يولد صور بدقة 2480x2480 مناسبة للطباعة
- **كل الأجزاء**: الأم، صفيحة واحدة، العنكبوت، الظهر، العضادة
